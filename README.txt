INICIALIZACION:
Abra el proyecto dentro de su editor de codigo.

Instale las dependencias mediante la terminal del editor de codigos:
npm i express

Inicilialize el proyecto:
npm run start

Abra el navegador en el puerto 3000:
http://localhost:3000/


Observaciones:
En caso de necesitar, instale nodejs:
https://nodejs.org/en/download/
